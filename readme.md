## Задание
- Написать функцию <b>sumOfArray</b>, которая возвращает число: сумму всех чисел массива <b>arr</b>.
- Функция принимает:
  - arr - массив чисел.

## Примечание
- Все элементы массивы - валидные числа.

## Примеры
- ( [1, 2, 3, 4, 5] ) => 15
- ( [] ) => 0
- ( [2, 4, 5] ) => 11
- ( [8, 7, 5, 2, 3, 1] ) => 26
